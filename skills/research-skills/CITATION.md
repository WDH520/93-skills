# 引用格式规范（GB/T 7714）

## 期刊论文

```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
[1] 张三, 李四. 深度学习方法研究[J]. 计算机学报, 2023, 46(5): 1001-1015.
```

## 会议论文

```
[序号] 作者. 题名[C]//会议名称. 地点: 出版者, 年: 起止页码.
[2] Wang S, Li M. A novel approach[C]//Proceedings of CVPR. Los Alamitos: IEEE, 2023: 100-110.
```

## 预印本

```
[序号] 作者. 题名[EB/OL]. (发布日期)[引用日期]. 网址.
[3] Brown T, et al. Language models are few-shot learners[EB/OL]. (2020-05-28)[2024-01-15]. https://arxiv.org/abs/2005.14165.
```

## 专著

```
[序号] 作者. 书名[M]. 版本(第1版不写). 出版地: 出版者, 年: 起止页码.
[4] 周志华. 机器学习[M]. 北京: 清华大学出版社, 2016: 50-80.
```

## 学位论文

```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.
[5] 王五. 基于深度学习的图像识别研究[D]. 北京: 中国科学院大学, 2022.
```
