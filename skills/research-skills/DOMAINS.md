## 1. 计算机科学/人工智能

### 1.1 核心数据库

| 数据库 | 适用场景 | 检索策略 | 替代方案（未配置MCP时） |
|--------|---------|---------|------------------------|
| **ArXiv** | 最新预印本、方法论文 | 类别: cs.CV, cs.LG, cs.AI, cs.CL, cs.RO | WebSearch: `site:arxiv.org "[主题]"` |
| **IEEE Xplore** | 会议论文、期刊 | 会议: CVPR, ICML, NeurIPS, ACL, ICCV | WebSearch: `site:ieeexplore.ieee.org "[主题]"` |
| **ACM Digital Library** | 期刊和会议 | 关键词 + 年份筛选 | WebSearch: `site:dl.acm.org "[主题]"` |
| **DBLP** | 文献索引、作者追踪 | 作者名 + 主题 | WebSearch: `site:dblp.org "[主题]"` |
| **Google Scholar** | 跨库检索、引用追踪 | 标题 + 作者组合 | 直接使用WebSearch |

### 1.2 典型评价指标

#### 准确性指标
| 指标 | 符号 | 定义 | 适用场景 |
|------|------|------|---------|
| 准确率 | Accuracy | (TP+TN)/(TP+TN+FP+FN) | 平衡数据集 |
| 精确率 | Precision | TP/(TP+FP) | 关注误报 |
| 召回率 | Recall | TP/(TP+FN) | 关注漏报 |
| F1分数 | F1-Score | 2×Precision×Recall/(Precision+Recall) | 综合性能 |
| AUC-ROC | AUC | ROC曲线下面积 | 分类阈值不确定 |

#### 效率指标
| 指标 | 单位 | 说明 |
|------|------|------|
| 推理时间 | ms | 单次前向传播时间 |
| FLOPs | G | 浮点运算次数 |
| 参数量 | M | 模型参数数量（百万） |
| 内存占用 | MB/GB | 运行时内存使用 |

#### 任务特定指标
| 任务 | 指标 | 说明 |
|------|------|------|
| 机器翻译 | BLEU | n-gram精确率加权几何平均 |
| 文本摘要 | ROUGE | 召回率导向的n-gram重叠 |
| 图像分割 | IoU, Dice | 预测与真实区域重叠度 |
| 目标检测 | mAP | 平均精度均值 |

### 1.3 文献结构模板

```
1. 引言
   - 1.1 问题背景
   - 1.2 技术挑战
   - 1.3 综述贡献
2. 背景知识
   - 2.1 基础概念
   - 2.2 相关技术
3. 方法分类
   - 3.1 传统方法
   - 3.2 深度学习方法
   - 3.3 最新进展
4. 实验对比
   - 4.1 数据集
   - 4.2 评价指标
   - 4.3 性能比较
5. 应用
6. 挑战与未来方向
7. 结论
```

### 1.4 术语规范

| 统一术语 | 避免使用 | 说明 |

### 2.2 典型评价指标

#### 临床诊断指标
| 指标 | 符号 | 定义 | 计算公式 |
|------|------|------|---------|
| 敏感性 | Sensitivity | 真阳性率 | TP/(TP+FN) |
| 特异性 | Specificity | 真阴性率 | TN/(TN+FP) |
| 阳性预测值 | PPV | 阳性结果中真阳性比例 | TP/(TP+FP) |
| 阴性预测值 | NPV | 阴性结果中真阴性比例 | TN/(TN+FN) |
| AUC-ROC | AUC | 诊断准确性 | ROC曲线下面积 |

#### 统计指标
| 指标 | 符号 | 说明 |
|------|------|------|
| 风险比 | HR | 两组事件发生率之比 |
| 比值比 | OR | 病例对照研究中暴露比值比 |
| 置信区间 | CI | 95% CI表示估计不确定性 |
| p值 | p | 统计显著性阈值通常<0.05 |

#### 生存分析指标
| 指标 | 说明 |
|------|------|
| 总生存期 | OS (Overall Survival) |
| 无进展生存期 | PFS (Progression-Free Survival) |
| 无病生存期 | DFS (Disease-Free Survival) |

### 2.3 文献结构模板（PRISMA风格）

```
1. 引言
   - 疾病背景/生物学背景
   - 研究现状
   - 综述目的
2. 方法
   - 2.1 检索策略（数据库、关键词、日期范围）
   - 2.2 纳入/排除标准
   - 2.3 质量评估方法
3. 结果
   - 3.1 文献筛选流程（PRISMA流程图）
   - 3.2 研究特征
   - 3.3 主要发现
4. 讨论
   - 4.1 证据总结
   - 4.2 局限性
   - 4.3 临床意义
5. 结论
```

## 8. 跨领域通用配置

### 8.1 通用数据库（所有领域）

| 数据库 | 优势 | 适用场景 |
|--------|------|---------|
| **Google Scholar** | 覆盖面广 | 跨学科检索、灰色文献 |
| **Web of Science** | 引文分析 | 高质量期刊、引文追踪 |
| **Scopus** | 综合学术 | 学科筛选、引文分析 |
| **ResearchGate** | 学者网络 | 获取全文、学者联系 |
| **Zotero** | 文献管理 | 本地管理、注释、PDF |

### 8.2 通用质量标准

| 标准 | 要求 |
|------|------|
| 引用规范 | 统一使用GB/T 7714格式 |
| 文献时效性 | 近5年文献占比≥60% |
| 文献质量 | 优先同行评审期刊 |
| 语言规范 | 术语一致性、语法正确性 |

### 8.3 通用结构元素

所有领域综述应包含：
- 标题页（标题、作者、单位）
- 摘要（背景、方法、结果、结论，200-300字）
- 关键词（3-6个）
- 引言（背景、目的、范围）
- 主体章节（按主题组织）
- 讨论（综合分析、局限性）
- 结论（总结、展望）
- 参考文献（80-120篇）

## 9. 领域自适应决策树

```
用户输入研究主题
    ↓
关键词匹配 → 识别主要领域
    ↓
选择对应数据库配置
    ↓
应用领域特定模板
    ↓
调整评价指标和术语规范
    ↓
生成定制化综述结构
```

### 9.1 多领域交叉处理

当研究主题涉及多个领域时：
1. 识别主要领域（权重最高）
2. 识别次要领域（辅助视角）
3. 融合多个领域的数据库
4. 采用主要领域的结构模板
5. 整合多领域的评价指标

### 9.2 交叉领域示例

| 主题 | 主要领域 | 次要领域 | 数据库组合 | 指标组合 |
|------|---------|---------|-----------|---------|
| 医学影像AI | 计算机科学/AI | 生物学/医学 | ArXiv + PubMed + IEEE | Accuracy + Sensitivity/Specificity |
| 教育技术 | 教育学 | 计算机科学/AI | ERIC + ArXiv | 学习效果 + 模型性能 |
| 环境经济 | 经济学 | 环境科学 | EconLit + Web of Science | 经济指标 + 环境指标 |
| 生物信息学 | 生物学 | 计算机科学 | PubMed + ArXiv | 生物学指标 + 计算效率 |

## 11. 数据库配置参考

### 11.1 MCP服务器配置（供用户参考）

#### ArXiv MCP配置
```json
{
  "mcpServers": {
    "arxiv": {
      "command": "uvx",
      "args": ["arxiv-mcp-server"],
      "env": {
        "ARXIV_STORAGE_PATH": "~/.arxiv-mcp-server/papers"
      }
    }
  }
}
```

#### PubMed MCP配置
```json
{
  "mcpServers": {
    "pubmedmcp": {
      "command": "uvx",
      "args": ["pubmedmcp@latest"],
      "env": {
        "UV_PRERELEASE": "allow",
        "UV_PYTHON": "3.12"
      }
    }
  }
}
```

### 11.2 WebSearch替代策略

当MCP工具未配置时，使用以下WebSearch策略：

| 数据库 | WebSearch查询模板 |
|--------|------------------|
| ArXiv | `site:arxiv.org "[主题]" [年份]` |
| PubMed | `site:pubmed.ncbi.nlm.nih.gov "[主题]"` |
| IEEE | `site:ieeexplore.ieee.org "[主题]"` |
| Google Scholar | `"[主题]" review survey filetype:pdf` |
