## 7. 参考文献格式模板（GB/T 7714）

### 7.1 期刊论文

```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.

示例：
[1] 张三, 李四. 深度学习方法研究[J]. 计算机学报, 2023, 46(5): 1001-1015.
[2] Wang S, Li M, Zhang L. A survey on transformer architecture[J]. IEEE Transactions on Pattern Analysis and Machine Intelligence, 2023, 45(3): 3001-3015.
```

### 7.2 会议论文

```
[序号] 作者. 题名[C]//会议名称. 地点: 出版者, 年: 起止页码.

示例：
[3] 王五, 赵六. 图像分类新方法[C]//中国计算机大会. 北京: 电子工业出版社, 2023: 150-160.
[4] Brown T, Mann B, Ryder N, et al. Language models are few-shot learners[C]//Advances in Neural Information Processing Systems 33. Vancouver: Curran Associates, 2020: 1877-1901.
```

### 7.3 预印本/电子文献

```
[序号] 作者. 题名[EB/OL]. (发布日期)[引用日期]. 网址.

示例：
[5] Smith J, Johnson K. A new approach to natural language processing[EB/OL]. (2023-05-15)[2024-01-20]. https://arxiv.org/abs/2305.12345.
```

### 7.4 专著/书籍

```
[序号] 作者. 书名[M]. 版本(第1版不写). 出版地: 出版者, 年: 起止页码.

示例：
[6] 周志华. 机器学习[M]. 北京: 清华大学出版社, 2016: 100-150.
[7] Goodfellow I, Bengio Y, Courville A. Deep learning[M]. Cambridge: MIT Press, 2016.
```

### 7.5 学位论文

```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.

示例：
[8] 陈七. 基于深度学习的图像识别研究[D]. 北京: 中国科学院大学, 2023.
```
