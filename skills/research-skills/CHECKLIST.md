# 质量检查清单

## 结构完整性
- [ ] 标题页和摘要
- [ ] 3-5个要点（Key Points）
- [ ] 引言（背景、挑战、贡献）
- [ ] 方法论章节
- [ ] 主要发现/方法章节
- [ ] 讨论（局限性、未来方向）
- [ ] 结论
- [ ] 参考文献（80-120篇）

## 内容质量
- [ ] 每个主要论点都有引用支持
- [ ] 每个方法类别都有局限性讨论
- [ ] 包含对比表格
- [ ] 包含图占位符
- [ ] 涵盖近期文献（近3-5年）
- [ ] 涵盖经典/奠基性文献

## 写作规范
- [ ] 使用谨慎措辞（hedging language）
- [ ] 术语使用一致
- [ ] 段落间过渡自然
- [ ] 引用格式统一（GB/T 7714）
