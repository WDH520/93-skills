# 通用文献综述撰写技能

一个支持任何学科领域的系统化文献综述撰写工具。

**版本**: 1.1.0  
**最后更新**: 2026-02-02

## 概述

本技能提供了一个完整的工作流，用于撰写高质量的学术文献综述，支持计算机科学、生物学、社会科学、医学、工程技术、经济学等多个学科领域。

## 特性

- **领域自适应系统** - 自动识别学科领域并调整撰写策略
- **7阶段结构化工作流** - 从文献收集到质量保障的完整流程
- **多学科模板支持** - 覆盖10+学科领域的专用模板
- **标准化写作规范** - 谨慎措辞、统一术语、GB/T 7714引用格式
- **质量控制清单** - 确保综述的学术标准
- **灵活的工具支持** - WebSearch默认支持，MCP工具可选配置

## 7阶段工作流

1. **领域识别与配置** - 自动检测学科领域，加载相应配置
2. **项目初始化** - 创建标准化项目结构和模板文件
3. **文献收集** - 根据领域选择数据库，系统检索文献
4. **大纲制定** - 确定章节结构，分配文献到各章节
5. **章节撰写** - 使用标准模板撰写各部分内容
6. **表格与图表** - 创建对比表格和图占位符
7. **质量保障** - 结构、内容、写作规范检查

## 领域自适应系统

### 自动领域检测

根据用户输入的研究主题，自动识别学科领域：

```
用户输入: "Transformer架构在自然语言处理中的应用综述"
    ↓
关键词匹配: Transformer, 架构, 自然语言处理
    ↓
检测领域: 计算机科学/AI (置信度: 高)
    ↓
加载配置: ArXiv数据库、方法分类模板、NLP术语规范
```

### 多领域交叉处理

对于跨领域主题（如医学影像AI）：
- 识别主要领域（方法导向）
- 识别次要领域（应用导向）
- 融合多领域数据库和指标
- 采用主要领域结构，整合次要领域要素

## 文献检索策略

### 方式1：WebSearch（默认，无需配置）

适用于所有学科领域：
- `"[主题] review"`
- `"[主题] survey"`
- `site:arxiv.org "[主题]"`
- `site:pubmed.ncbi.nlm.nih.gov "[主题]"`

### 方式2：ArXiv MCP（可选配置）

适用领域：计算机科学、物理学、数学
- 配置方法：添加arxiv-mcp-server到MCP配置
- 未配置时：使用WebSearch的site:arxiv.org替代

### 方式3：PubMed MCP（可选配置）

适用领域：医学、生物学、生命科学
- 配置方法：添加pubmedmcp到MCP配置
- 未配置时：使用WebSearch的site:pubmed替代

```markdown
## 3. 研究方法/理论框架
### 3.1 [类别1]
### 3.2 [类别2]

## 4. 应用与发现

## 5. 实践意义/政策影响

## 6. 讨论
### 6.1 当前局限性
### 6.2 未来方向

## 7. 结论

## 参考文献（GB/T 7714格式）
```

## 引用格式规范（GB/T 7714）

### 期刊论文
```
[1] 张三, 李四. 深度学习方法研究[J]. 计算机学报, 2023, 46(5): 1001-1015.
```

### 会议论文
```
[2] Wang S, Li M. A novel approach[C]//Proceedings of CVPR. Los Alamitos: IEEE, 2023: 100-110.
```

### 预印本
```
[3] Brown T, et al. Language models are few-shot learners[EB/OL]. (2020-05-28)[2024-01-15]. https://arxiv.org/abs/2005.14165.
```

## 写作规范

### 谨慎措辞
- 使用"可能"、"表明"、"似乎"、"显示出有希望的结果"
- 避免绝对性断言（不要说"X是最好的方法"）

### 引用规范
- 每个主要论点都需要引用支持
- 引用格式统一使用GB/T 7714
- 近5年文献占比≥60%

### 必备要素
- **要点框**（3-5个要点）在标题后
- 每个主要章节的**对比表格**
- 统一的**性能指标**格式（根据领域调整）
- **图占位符**及详细说明
- 按主题组织的**参考文献**（通常80-120篇）

## 使用示例

### 示例1：计算机科学

```
系统响应:
- 识别领域: 计算机科学/AI
- 推荐数据库: ArXiv, IEEE Xplore, ACM
- 生成结构: 引言→背景→GNN方法→推荐应用→实验对比→挑战→结论
- 创建文件: CLAUDE.md, IMPLEMENTATION_PLAN.md, manuscript_draft.md
```

### 示例2：医学
```
用户: 需要一篇关于"免疫检查点抑制剂在肺癌治疗中应用"的综述
```

## 错误处理与回退策略

| 错误场景 | 处理策略 |
|---------|---------|
| 领域检测失败 | 使用通用配置，询问用户确认 |
| MCP工具未配置 | 使用WebSearch替代方案 |
| 文献数量不足 | 扩展关键词，放宽日期限制 |
| 检索结果质量低 | 增加限定词，限定高质量来源 |
