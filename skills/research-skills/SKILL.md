# 通用文献综述撰写技能

系统化的文献综述撰写工作流，支持任何学科领域的学术研究。

## 快速开始

当用户请求文献综述时：

1. **领域识别** - 使用下方"领域自动检测规则"识别学科领域
2. **初始化项目** - 在当前工作目录创建三个核心文件：
   - `CLAUDE.md` - 写作指南和术语规范
   - `IMPLEMENTATION_PLAN.md` - 分阶段执行计划
   - `manuscript_draft.md` - 主文稿
3. **遵循7阶段工作流** - 详见 [WORKFLOW.md](WORKFLOW.md)
4. **使用标准模板** - 详见 [TEMPLATES.md](TEMPLATES.md)

## 领域自动检测规则

根据用户输入的研究主题自动识别学科领域：

### 检测逻辑

1. 提取用户输入中的关键词
2. 匹配 [DOMAINS.md](DOMAINS.md) 中的领域关键词表
3. 计算匹配度（匹配词数/总词数）
4. 选择匹配度最高的领域为主要领域
5. 如匹配度<50%，询问用户确认

### 多领域交叉处理

当检测到多个领域关键词时：
1. 确定主要领域（匹配度最高的）
2. 确定次要领域（匹配度次高的，且>30%）
3. 融合两个领域的数据库配置和评价指标
4. 采用主要领域的结构模板，整合次要领域要素

**常见交叉领域示例**：
- 医学影像AI：主要=计算机科学/AI，次要=生物学/医学
- 教育技术：主要=教育学，次要=计算机科学/AI
- 环境经济：主要=经济学，次要=环境科学

> **完整领域配置**（数据库、评价指标、术语规范）：详见 [DOMAINS.md](DOMAINS.md)

## 核心原则

### 写作风格
- 使用**谨慎措辞**："可能"、"表明"、"似乎"、"显示出有希望的结果"
- 避免绝对性断言：不要说"X是最好的方法"
- 每个论点都需要引用支持
- 每个主要章节都需要**局限性**段落

### 必备要素
- **要点框**（3-5个要点）在标题后
- 每个主要章节的**对比表格**
- 统一的**性能指标**格式（根据领域调整）
- **图占位符**及详细说明
- 按主题组织的**参考文献**（通常80-120篇）
- 引用格式统一使用**GB/T 7714**标准

### 段落结构
1. 主题句（主要论点）
2. 支持证据（引用+数据）
3. 分析（批判性评价）
4. 过渡到下一段

## 文献来源

### 工具使用策略

**方式1：WebSearch（默认，无需配置）**
- 适用场景：所有学科领域
- 搜索策略：`"[主题] review"`、`"[主题] survey"`、`site:arxiv.org "[主题]"`

**方式2：ArXiv MCP（需用户配置）**
- 适用领域：计算机科学、物理学、数学、定量生物学
- 如果未配置，使用WebSearch搜索`site:arxiv.org`替代

**方式3：PubMed MCP（需用户配置）**
- 适用领域：医学、生物学、生命科学
- 如果未配置，使用WebSearch搜索`site:pubmed.ncbi.nlm.nih.gov`替代

### 各领域推荐数据库

详见 [DOMAINS.md](DOMAINS.md) 获取完整配置：

| 领域 | 主要数据库 | 次要数据库 |
|------|-----------|-----------|
| 计算机科学/AI | ArXiv, IEEE Xplore | ACM, DBLP |
| 生物学/医学 | PubMed, PMC | Embase, Cochrane |
| 社会科学 | JSTOR, Web of Science | PsycINFO, ERIC |
| 工程技术 | IEEE Xplore, EI | ASME, AIAA |
| 经济学 | EconLit, RePEc | SSRN, JSTOR |
