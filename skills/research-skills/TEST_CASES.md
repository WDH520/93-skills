# 多学科测试用例

本文件包含用于验证通用文献综述技能模块在不同学科领域适用性的测试用例。

## 测试用例1：计算机科学领域

### 测试主题
**"Transformer架构在自然语言处理中的应用综述"**

### 预期领域检测
- **主要领域**: 计算机科学/人工智能
- **置信度**: 高
- **关键词匹配**: Transformer、架构、自然语言处理

### 预期数据库配置
| 数据库 | 优先级 | 预期检索数量 |
|--------|--------|-------------|
| ArXiv | 高 | 50-100篇 |
| IEEE Xplore | 高 | 30-50篇 |
| ACM Digital Library | 中 | 20-30篇 |
| Google Scholar | 中 | 补充检索 |

### 预期结构模板
```
1. 引言
2. 背景知识（注意力机制、序列建模）
3. Transformer架构变体
   - 3.1 原始Transformer
   - 3.2 BERT系列
   - 3.3 GPT系列
   - 3.4 其他变体（T5, XLNet等）
4. 应用任务
   - 4.1 文本分类
   - 4.2 机器翻译
   - 4.3 问答系统
   - 4.4 文本生成
5. 性能评估
6. 挑战与未来方向
7. 结论
```

### 预期评价指标
- **准确性**: Accuracy, F1-Score, BLEU, ROUGE
- **效率**: 推理时间、参数量、FLOPs
- **可扩展性**: 序列长度处理能力

### 验证检查清单
- [ ] 正确识别为计算机科学领域
- [ ] 推荐ArXiv和IEEE Xplore作为主要数据库
- [ ] 结构模板包含方法分类和性能评估章节
- [ ] 术语规范包含NLP领域标准术语
- [ ] 表格模板包含模型性能对比表

## 测试用例2：生物学领域

### 测试主题
**"CRISPR-Cas9基因编辑技术在癌症治疗中的研究进展"**

### 预期领域检测
- **主要领域**: 生物学/医学
- **置信度**: 高
- **关键词匹配**: CRISPR、基因编辑、癌症、治疗

### 预期数据库配置
| 数据库 | 优先级 | 预期检索数量 |
|--------|--------|-------------|
| PubMed | 高 | 50-80篇 |
| PubMed Central | 高 | 30-50篇 |
| Web of Science | 中 | 20-30篇 |
| Cochrane Library | 中 | 临床试验相关 |

### 预期结构模板
```
1. 引言
2. 方法
   - 2.1 检索策略
   - 2.2 纳入/排除标准
   - 2.3 质量评估
3. 结果
   - 3.1 文献筛选流程
   - 3.2 CRISPR机制概述
   - 3.3 癌症类型应用
     - 血液肿瘤
     - 实体瘤
   - 3.4 临床试验进展
4. 讨论
   - 4.1 安全性与脱靶效应
   - 4.2 递送系统挑战
   - 4.3 伦理与监管
   - 4.4 未来方向
5. 结论
```

### 预期评价指标
- **临床指标**: 客观缓解率(ORR)、无进展生存期(PFS)、总生存期(OS)
- **安全指标**: 不良事件发生率、脱靶率
- **分子指标**: 基因编辑效率、特异性

### 验证检查清单
- [ ] 正确识别为生物学/医学领域
- [ ] 推荐PubMed作为主要数据库
- [ ] 结构模板遵循PRISMA指南风格
- [ ] 术语规范包含医学和基因编辑术语
- [ ] 表格模板包含临床试验特征表

- **关键词匹配**: 社交媒体、青少年、心理健康、影响

- **次要领域**: 生物学/医学（应用导向）
- **置信度**: 高
- **关键词匹配**: 深度学习、医学影像、诊断

| 数据库 | 优先级 | 预期检索数量 |
|--------|--------|-------------|
| ArXiv | 高 | 50-80篇（方法） |
| PubMed | 高 | 40-60篇（临床） |
| IEEE Xplore | 高 | 30-50篇 |
| Google Scholar | 中 | 交叉领域补充 |

### 预期结构模板
```
1. 引言
2. 背景
   - 2.1 医学影像概述
   - 2.2 深度学习基础
3. 方法分类
   - 3.1 分类方法
   - 3.2 检测方法
   - 3.3 分割方法
4. 应用领域
   - 4.1 放射影像
   - 4.2 病理影像
```

### 验证检查清单
